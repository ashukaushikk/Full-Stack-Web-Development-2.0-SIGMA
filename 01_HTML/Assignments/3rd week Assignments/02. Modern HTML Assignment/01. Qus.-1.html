About HTML5

HTML5 is far more dynamic and incorporates multimedia components than previous versions of HTML, which enables you to design mostly static web pages that need to be spiced up with CSS and JavaScript. HTML5 features list mentioned below:

  • Video and Audio Features
  • Header and Footer
  • These are the new values that the input tag may include:
    • Section
    • Main
  • Figure and figcaption
  • Placeholders
  • Canvas
