Most common Formatting tags

  1. "h1-h6"
    • Use "h1" for the main heading of a page, and use subsequent headings for subheadings in a hierarchical order.
    • Use headings to create a clear and concise outline of the content of the page.
    • Use headings to emphasize important points or sections of the page.
    • Don't use heading tags purely for styling purposes. Use CSS to style headings instead.
    • Use only one "h1" tag per page, as it should represent the main topic of the page.
  2. "audio"
    • The "audio" tag is used to embed sound content in a document, such as music or other audio streams.
    • The "audio" tag contains one or more <"source"> tags with different audio sources. The browser will choose the first source it supports.
    • The text between the <"audio"> and <"/audio"> tags will only be displayed in browsers that do not support the <"audio"> element.
  3. "dd"
    • The <"dd"> tag is used to describe a term/name in a description list.
    • The <"dd"> tag is used in conjunction with <"dl"> (defines a description list) and <"dt"> (defines terms/names).
    • Inside a <"dd"> tag you can put paragraphs, line breaks, images, links, lists, etc.
  4. "div"
    • The <"div"> tag defines a division or a section in an HTML document.
    • The <"div"> tag is easily styled by using the class or id attribute.
    • The <"div"> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
    • Any sort of content can be put inside the <"div"> tag!
  5. "img"
    • The <"img"> tag is used to embed an image in an HTML page.
    • Images are not technically inserted into a web page; images are linked to web pages. The <"img"> tag creates a holding space for the referenced image.
    • The <"img"> tag has two required attributes:
      • src - Specifies the path to the image
      • alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed
