Semantic tag

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. The following HTML tags can be used to break your page into identified parts:

  1. <'header'>: t defines a header for a web page.
  2. <'nav'>: It defines a container for navigation links.
  3. <'section'>: This defines a section in a web page.
  4. <'article'>: This element contains the main part, containing information about the web page.
  5. <'aside'>: The <'aside'> content is often placed as a sidebar in a document.
  6. <'footer'>: It defines a footer for a document or a section.
